# intro-to-rails

## Overview
This is a practice project for learning Ruby on Rails. The primary goal is to develop a simple web application that showcases the capabilities of Rails.

## Features
- User authentication
- CRUD operations
- Responsive design

## Tech Stack
- Ruby on Rails
- PostgreSQL
- Bootstrap

## Getting Started
To get started, clone the repository and run `bundle install` to install dependencies.

## Future Plans
- Add more features like social login.
- Improve the UI with modern design principles.
